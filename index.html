Windows Defender / Real Time protection & Antivirous off karke hi Updater Run kare .

Windows Defender ke real-time protection ko band karne ke liye step-by-step guide neeche diya hai. Ye steps Windows 10 ya Windows 11 ke liye kaam karte hain. Dhyan rakhein ki real-time protection band karna temporary hona chahiye kyunki isse system ka security risk badh sakta hai.

Steps to Turn Off Windows Defender Real-Time Protection:

Settings Ko Open Karein
Start menu par click karein (Windows icon neeche left corner mein).
"Settings" (gear icon) par click karein ya search bar mein "Settings" type karke enter press karein.
Update & Security Mein Jayein
Settings window mein "Update & Security" option par click karein.
Windows Security Section Mein Jayein
Left side ke menu mein "Windows Security" select karein.
Right side par "Open Windows Security" button par click karein.
Virus & Threat Protection Ko Select Karein
Windows Security window khulne ke baad, "Virus & Threat Protection" option par click karein.
Manage Settings Par Jayein
Virus & Threat Protection section mein, "Manage Settings" link par click karein (ye "Virus & Threat Protection Settings" ke neeche hoga).
Real-Time Protection Ko Band Karein
"Real-Time Protection" ke neeche ek toggle switch hoga jo "On" position mein hoga.
Is switch par click karein aur ise "Off" position mein karein.
Windows User Account Control (UAC) prompt aaye to "Yes" par click karein.
Confirmation Check Karein
Real-Time Protection ab band ho chuka hoga. Aapko ek warning message dikh sakta hai ki system ab protected nahi hai.
Dhyan dein ki Windows Defender real-time protection ko automatically kuch samay baad wapas on kar sakta hai.
